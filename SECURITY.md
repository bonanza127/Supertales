# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in Supertales, please report it by opening a [security advisory](https://github.com/bonanza127/Supertales/security/advisories/new) or emailing the maintainer directly.

Please do **not** report security vulnerabilities through public GitHub issues.

We aim to acknowledge receipt within 48 hours and provide a fix within 7 days depending on severity.
